Pyros is a software layer on top of the ROS framework, allowing you to interface your ROS systems with : - standard python dev dynamic workflow (no generation, compilation time wasted) - standard python packages from PyPI, leveraging NNNNN packages - any existing Cloud REST Services (via HTTP) - any existing Frontend Application and GUI (via HTTP) - any existing IoT infrastructure (via MQTT, RIOT, etc.)
